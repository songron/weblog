weblog
======

个人博客。

Download
--------
    $ git clone https://github.com/ghostrong/weblog.git
    
Quick Start
-----------

- 配置数据库连接参数：config.py
- 创建table

    $ python data.py
- 运行 app

    $ python myapp.py
